# Ургамлын Өвчнийг Илрүүлэх deep learning (CNN)

**Convolutional Neural Network (CNN)** ашиглан ургамлын өвчнийг илрүүлж, ангилах юм. Модель нь өгөгдөлд **data augmentation** хийж сурдаг ба шалгалтын багцаар үнэлэгддэг. Мөн ганц зураг дээр таамаг гаргах боломжтой.
